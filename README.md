# 表情宇宙 / Emotion Universe

单页静态 Web 应用：用摄像头捕捉面部动作与表情，实时驱动 Three.js 粒子笑脸世界（本地处理，不上传、不存储）。

## 本地运行

摄像头在大多数浏览器里需要 HTTPS 或 `localhost`：

```bash
npm run dev
```

打开 `http://localhost:5173`。

## URL 参数

- `?quality=high|medium|low`：强制性能档位（覆盖 Auto 选择）

## 兼容性提示

- 微信内置浏览器对摄像头/全屏限制更常见：建议“在浏览器打开”。
- iPad Pro 1 代/低性能设备默认走 `Low`，也可在面板里手动切换。

## 交互要点

- 微笑 / 张嘴 / 闭眼 / 摇头 / 点头会驱动笑脸粒子的形变与运动
- 连续摇头会触发粒子颜色循环变化（仍可在面板里选基础色）
